```bash
npm install -g agentsys && agentsys
```

Select your platform, pick your plugins, and start working.

## Philosophy

- **Detection uses code, judgment uses AI** -- regex, AST analysis, and static analysis handle deterministic work. LLM calls handle synthesis, planning, and review.
- **One agent, one job** -- each agent has a single responsibility, a specific model assignment, and defined inputs/outputs.
- **Pipeline with gates** -- each step must pass before the next begins. Agents cannot skip phases.
- **Composable, not monolithic** -- every command works standalone. Together, they compose into end-to-end pipelines.
- **Persistent state** -- workflow state survives session boundaries. Resume from any point.
- **Right model for the task** -- Opus for reasoning, Sonnet for validation, Haiku for mechanical execution.
